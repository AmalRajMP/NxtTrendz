# NxtTrendz

An e-commerce web application where users can browse products, manage a shopping cart, and place orders with authentication and protected routes.

## Key Features
- Authentication with protected routes
- Product listing with filters and sorting
- Add to cart with quantity management
- Cart summary with total items and total cost
- Cart management using Context API
- Responsive design
